    01. Introduction

    The Client

    Ultra Cleaners is a family-owned drycleaning business based in Scottsdale, Arizona, USA. They have been in business since 1995 but this is their first official website. I was hired to design a simple website to represent their store. The only requirement of the owner was to use their business colors ( pink and purple) and everything else was up to my style and creativity.

    My Role: Web Designer, UI Designer, Graphic Designer, Front-end Developer

    Project Date: 2017. June

    Platforms: Responsive

    03. WireFrame

    At first I created the website's wireframe. I was focusing on showcasing the website's layout, the hierarchy of the content and the interaction of the user interface. The flat layout is easy to navigate on - by just scrolling up and down - so the user doesn't get lost in the unnecessary many pages. The sticky navigation bar on the top helps the visitor guide himself quickly on the website.

    The main goal was to faciliate efficiency and usability as much as possible with a simple and very clean look.

    04.Responsive Website Design

    Website Landing Page

    With the smart use of proportion-based grids, flexible images and CSS3 media queries the website is optimal for viewing on any device. Improves usability and accessability. With responsive design the layout of the page can be customized so that it is easy on the eyes whatever device the user views the site on.

    I developed the website from scratch with the use of Sublime Text code editor. Twitter Bootstrap Front-end Framework was my biggest help to create a visually-coherent website that is responsive and cross-browser compatible.
